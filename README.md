# MonkeyBrain
### History
on 3rd year of my UNI I had a Cryptography course, which involved doing several little projects (6, to be exact). At the beginning of the course I had decided that I wanted to make the task a bit more difficult. Instead of 6 little CLI projects I wanted to make one "app" with simple GUI that has 6 tabs (each representing 1 function). 
After some time I've thought that as for beginner, this was a github worthy project and the tasks were pretty fun, so now it's sitting here.

**This is also my first github posted (or whatever the correct term for this) project.**



### Functionality
the app has 6 tabs each representing one cryptography related method. They are isolated and do not contact/communicate in any way
- tab 1: simple Ceasars code. (UA and EN alphabet). Ability to encode and decode messages and set the shift value
- tab 2: not documented. Similar kind of thing. Can only ENCODE and check the number of mixing.
- tab 3: not really cipher, but tools to decipher Ceasard encoded messages. Uses plots for analytic info
- tab 4: steganography on BPM file (todo)
- tab 5: in progress
- tab 6: in progress
